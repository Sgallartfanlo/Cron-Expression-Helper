# Cron Expression Helper

> Creador i explicador d'expressions cron.

Construeix expressions cron visualment, tradueix-les a text llegible, consulta les pròximes execucions i copia-les per als teus scripts i crontab.

## Tecnologies

- HTML + CSS + JavaScript (vanilla)
- [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- [Font Awesome](https://fontawesome.com/) (icons)
- [Fira Code](https://github.com/tonsky/FiraCode) + [Inter](https://rsms.me/inter/) fonts

## Funcionalitats

- Constructor visual amb camps individuals (minut, hora, dia, mes, dia setmana)
- Més de 10 presets ràpids (cada minut, cada hora, cada dia, feiners, etc.)
- Explicació en text llegible de l'expressió
- Càlcul de pròximes 5 execucions
- Suport de @reboot, @daily, @weekly, @monthly
- Copiar expressió al portapapers
- Guia de sintaxi bàsica

## Ús

Obre `cron-expression-helper.html` al navegador, ajusta els camps individuals o selecciona un preset, i copia l'expressió generada.
