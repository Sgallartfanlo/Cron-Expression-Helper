Cron Expression Helper

Creador i explicador d'expressions Cron

Construeix expressions cron visualment, tradueix-les a text llegible i copia-les per als teus scripts i crontab.

Constructor visual

Expressió cron

* * * * *

Presets ràpids

Explicació

Cada minut de cada hora, cada dia, cada mes, cada dia de la setmana.

Pròximes execucions:

crontab 
# ┌───────── minut (0-59)
# │ ┌──────── hora (0-23)
# │ │ ┌─────── dia del mes (1-31)
# │ │ │ ┌────── mes (1-12)
# │ │ │ │ ┌───── dia de la setmana (0-7, 0=dg)
# │ │ │ │ │
# * * * * *

Consells

  • * = qualsevol valor
  • */n = cada n intervals
  • a-b = rang de valors
  • a,b,c = llista de valors
  • @reboot = en arrencar
  • @daily = 0 0 * * *
  • @weekly = 0 0 * * 0
  • @monthly = 0 0 1 * *
